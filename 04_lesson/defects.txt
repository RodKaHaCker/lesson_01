=== БАГ-РЕПОРТ №1 ===
Заголовок: Метод contains() возвращает True для пустой строки и пустого символа.
Описание: При вызове contains("", "") метод возвращает True, хотя ожидается False.
Ожидаемый результат: False.
Фактический результат: True.
Серьёзность: Низкая.
Шаги воспроизведения:
1. Создать экземпляр StringUtils.
2. Вызвать метод contains("", "").

=== БАГ-РЕПОРТ №2 ===
Заголовок: Метод contains() не обрабатывает None.
Описание: При передаче None в качестве первого аргумента возникает ошибка AttributeError.
Ожидаемый результат: Метод должен возвращать False.
Фактический результат: Выбрасывается исключение AttributeError.
Серьёзность: Средняя.
Шаги воспроизведения:
1. Создать экземпляр StringUtils.
2. Вызвать метод contains(None, "a").
